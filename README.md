# 🛒 Walmart Management System

## 📌 Project Overview

The **Walmart Management System** is a web-based application developed to simulate the operations of a large-scale retail chain like Walmart.
This system helps in managing products, customers, orders, and inventory efficiently using a structured and scalable approach.

The project is built using **Python and Django framework**, enabling dynamic data handling and smooth user interaction.

---

## 🎯 Objectives

* To develop a system that manages Walmart store operations
* To automate product, order, and inventory management
* To provide a centralized platform for handling retail data
* To understand real-world application of backend frameworks

---

## ⚙️ Features

* 🛍️ Product Management (Add, Update, Delete Products)
* 📦 Inventory Tracking
* 👤 Customer Management
* 🧾 Order Processing System
* 🔍 Product Search Functionality
* 📊 Data Management using Django Models
* 🌐 Web-based Interface

## 🚀 How to Run the Project

1. Clone the repository:

```id="gq7xk1"
git clone https://github.com/your-username/walmart-project.git
```

2. Navigate to project folder:

```id="9b7zch"
cd walmart-project
```

3. Install dependencies:

```id="y3q6kq"
pip install django
```

4. Run migrations:

```id="p9i0eo"
python manage.py migrate
```

5. Start the server:

```id="z0q4jv"
python manage.py runserver
```

6. Open browser:

```id="s5p9un"
http://127.0.0.1:8000/
```
